Liminall Node on fyysinen tunniste. Se kiinnitetään mihin tahansa pintaan. Koskettamalla sitä puhelimella avautuu omistajan määrittelemä sivu. Liminall Node is a physical marker. It can be attached to any surface. Touching it with a phone opens the page defined by the owner.

Tunniste ei seuraa. Ei paikanna. Ei kerää tietoa. Se herää vain kosketuksesta. The marker does not track. It does not locate. It does not collect data. It activates only through touch.

Sisältöä voi muuttaa milloin tahansa. Tunniste pysyy samana. The content can be changed at any time. The marker remains the same.

Liminall Node koostuu kahdesta osasta: fyysisestä tunnisteesta ja siihen kytketystä digitaalisesta näkymästä. Liminall Node consists of two parts: a physical marker and a connected digital view.

Fyysinen tunniste kiinnitetään esineeseen, tilaan tai pintaan. Se on pysyvä - se ei vaihdu vaikka sisältö muuttuisi. Digitaalinen näkymä on omistajan hallinnassa. Sitä voi päivittää milloin tahansa: tekstiä, ohjeita, yhteystietoja, kuvia, statusta. The physical marker is attached to an object, space, or surface. It is permanent - it does not change even if the content changes. The digital view is controlled by the owner and can be updated at any time: text, instructions, contact details, images, status.

Nämä kaksi ovat erillisiä mutta sidoksissa toisiinsa. Tunniste on avain. Näkymä on ovi. Omistaja päättää mihin ovi johtaa. These two are separate but connected. The marker is the key. The view is the door. The owner decides where the door leads.

Liminall Nodessa on myös hallinta-avain - erillinen fyysinen tunniste, joka avaa muutoskanavan. Julkinen tunniste näyttää sisällön. Hallinta-avain muuttaa sitä. Liminall Node also includes a control key - a separate physical marker that opens the modification channel. The public marker displays the content. The control key changes it.

Näitä ei voi sekoittaa keskenään, eikä hallintaa voi kaapata verkon kautta. These cannot be mixed, and control cannot be taken over through the network.

Liminall Node käyttää NTAG213-siruteknologiaa tai vaihtoehtoisesti suojattuja variantteja vaativampiin käyttökohteisiin. Liminall Node uses NTAG213 chip technology or protected variants for more demanding use cases.

Tunniste on yhteensopiva kaikkien NFC-yhteensopivien laitteiden kanssa ilman erillistä sovellusta. iPhone 7 ja uudemmat sekä suurin osa Android-laitteista vuodesta 2016 eteenpäin tukevat lukemista suoraan käyttöjärjestelmätasolla. The marker is compatible with all NFC-enabled devices without an additional app. iPhone 7 and newer, and most Android devices from 2016 onward support reading at the OS level.

NFC toimii lyhyellä etäisyydellä (1-4 cm). Tunniste ei lähetä signaalia itse - se vastaa vain, kun puhelin luo kentän. NFC works at a short range (1-4 cm). The tag does not transmit a signal by itself - it responds only when the phone creates the field.

Tunniste kestää UV-säteilyä ja säänvaihteluita. Se on saatavilla eri alustoilla: polymeeri, metalli, komposiitti. On-metal-versiot toimivat metallipinnoilla ilman signaalin heikkenemistä. The marker withstands UV exposure and weather conditions. It is available on different substrates: polymer, metal, composite. On-metal versions work on metal surfaces without signal loss.

Dual-Anchor-rakenne erottaa julkisen tunnisteen ja hallinta-avaimen toisistaan fyysisellä tasolla. Muutoksia ei tehdä selainistunnon kautta eikä niihin pääse käsiksi linkin tai verkon kautta. Omistajuus on fyysinen, ei kirjautumispohjainen. The Dual-Anchor structure separates the public marker and the control key at a physical level. Changes are not made through browser sessions and cannot be accessed via links or networks. Ownership is physical, not login-based.
