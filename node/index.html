Node Node

Liminall Node on fyysinen tunniste. Liminall Node is a physical marker.

Koskettamalla sitä puhelimella avautuu omistajan määrittelemä sivu. Touching it with a phone opens the page defined by the owner.

Tunniste ei seuraa, paikanna tai kerää tietoa. Se herää vain kosketuksesta. The marker does not track, locate, or collect data. It wakes only through touch.

Sisältöä voi muuttaa milloin tahansa. Tunniste pysyy samana. The content can be changed at any time. The marker remains the same.

Liminall Node koostuu kahdesta osasta: fyysisestä tunnisteesta ja siihen kytketystä digitaalisesta näkymästä. Liminall Node consists of two parts: a physical marker and the connected digital view.

Tunniste on pysyvä. Näkymä on muokattavissa. The marker is permanent. The view is editable.

Tunniste on avain. Näkymä on ovi. The marker is the key. The view is the door.

Omistaja päättää mihin ovi johtaa. The owner decides where the door leads.

Hallinta tehdään erillisellä fyysisellä avaimella. Julkinen tunniste näyttää sisällön. Hallinta-avain muuttaa sitä. Control is handled with a separate physical key. The public marker shows the content. The control key changes it.

Liminall Node käyttää NFC-siruteknologiaa (esim. NTAG213). Liminall Node uses NFC chip technology (for example NTAG213).

Se toimii ilman sovellusta kaikissa NFC-yhteensopivissa laitteissa. It works without an app on all NFC-compatible devices.

Tunniste kestää ympäristöolosuhteita ja on saatavilla eri materiaaleissa. The marker withstands environmental conditions and is available in different materials.

Hallinta ei tapahdu verkon kautta. Omistajuus on fyysinen, ei kirjautumispohjainen. Control does not happen through the web. Ownership is physical, not login-based.
