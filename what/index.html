Liminall

Mikä tämä on What is this

Liminall Node liittää kohteen suoraan verkkoon. Liminall Node connects a target directly to the web.

Kosketus avaa yhteyden ilman sovellusta tai kirjautumista. A touch opens the connection without an app or login.

Yhteys avaa kohteen oman sivun. The connection opens the target’s own page.
